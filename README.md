# SectorHLabs 🚀

¡Hola! Somos **SectorHLabs**, un estudio de desarrollo de software enfocado en crear experiencias digitales innovadoras, desde aplicaciones móviles hasta motores de juego.

## 🛠️ Lo que hacemos

En SectorHLabs nos apasiona la tecnología y el desarrollo multiplataforma. Nuestros proyectos actuales abarcan:

- 🎮 **Game Development**: Creación de motores y experiencias de juego (Proyecto: *Gallero*).
- 📱 **Mobile & Web Apps**: Aplicaciones modernas y escalables con Next.js, React y Capacitor (Proyectos: *VocApp*, *Recetas Mamá*).
- ⚙️ **Software Tools**: Herramientas y utilidades especializadas (Proyecto: *LEGRITE*).

## 🧰 Stack Tecnológico

- **Frontend**: React, Next.js, Tailwind CSS, TypeScript.
- **Mobile**: Capacitor, Flutter.
- **Backend**: Node.js, Python, Java.
- **Game Dev**: Motores propios y librerías especializadas.
